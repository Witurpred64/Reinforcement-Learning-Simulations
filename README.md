
# Reinforcement Learning Simulations

This repository hosts implementations and simulations of various Reinforcement Learning (RL) algorithms. It covers fundamental concepts, classic algorithms like Q-learning and SARSA, and more advanced policy-gradient methods such as DQN, PPO, and A2C.

## Algorithms Implemented
- **Value-Based Methods:** Q-Learning, SARSA, Deep Q-Networks (DQN).
- **Policy-Based Methods:** REINFORCE, Actor-Critic (A2C).
- **Model-Based Methods:** (Future additions)

## Environments
- OpenAI Gym (CartPole, LunarLander, etc.)
- Custom simulation environments.

## Technologies
- PyTorch
- Gymnasium (formerly OpenAI Gym)
- NumPy

## Getting Started
Each algorithm implementation resides in its own directory. Refer to the respective `README.md` files for installation and execution instructions.
